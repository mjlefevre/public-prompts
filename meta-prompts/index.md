# Role-Based Meta-Prompts

## QA (Quality Assurance)

1. "Help me improve this QA prompt [prompt] so it produces specific, actionable test scenarios instead of generic advice."
2. "Rewrite this prompt [prompt] to make the model ask clarifying questions before generating a test plan."
3. "Create a reusable prompt template for generating risk-based test plans for [feature/release]."
4. "Improve this bug-reporting prompt [prompt] so the output includes reproduction steps, expected behavior, actual behavior, evidence, and impact."
5. "Turn this vague QA request [request] into a precise prompt with role, context, inputs, output format, and quality criteria."
6. "Create a prompt that helps QA identify missing acceptance criteria and ambiguous requirements in [story/doc]."
7. "Revise this regression planning prompt [prompt] so it prioritizes by risk, historical defects, and changed areas."
8. "Create a prompt template for converting exploratory testing notes into defects, questions, risks, and follow-up tests."
9. "Evaluate this QA prompt [prompt] and list what context, constraints, or output structure would improve the answer."
10. "Write a prompt that instructs the model to critique a test strategy before suggesting improvements."

## Business Analysts

1. "Help me improve this BA prompt [prompt] so it elicits complete requirements, assumptions, risks, and open questions."
2. "Rewrite this stakeholder interview prompt [prompt] to avoid leading questions and uncover real needs."
3. "Create a reusable prompt template for turning business goals into capabilities, requirements, dependencies, and delivery order."
4. "Improve this process analysis prompt [prompt] so the output separates current state, pain points, future state, and gaps."
5. "Turn this vague discovery request [request] into a precise prompt with audience, context, artifacts, and output format."
6. "Create a prompt that helps identify ambiguous, non-testable, or conflicting requirements in [doc]."
7. "Revise this user story prompt [prompt] so it produces acceptance criteria, NFRs, dependencies, and definition of ready checks."
8. "Create a prompt template for converting workshop notes into decisions, action items, risks, and open questions."
9. "Evaluate this BA prompt [prompt] and recommend missing context, better constraints, and clearer output structure."
10. "Write a prompt that asks the model to challenge a proposed solution before generating requirements."

## Developers

1. "Help me improve this coding prompt [prompt] so it includes language, framework, constraints, expected behavior, tests, and output format."
2. "Rewrite this debugging prompt [prompt] to provide the model with reproduction steps, logs, code, environment, and hypotheses."
3. "Create a reusable prompt template for asking the model to implement [feature] with tests and documentation."
4. "Improve this refactoring prompt [prompt] so it preserves behavior, limits scope, and requires a verification plan."
5. "Turn this vague technical request [request] into a precise prompt with context, constraints, files, expected output, and acceptance criteria."
6. "Create a prompt that helps generate SQL, regex, or shell commands with examples, safety notes, and explanations."
7. "Revise this code review prompt [prompt] so it prioritizes bugs, regressions, security risks, and missing tests."
8. "Create a prompt template for explaining unfamiliar code to a developer onboarding to [codebase]."
9. "Evaluate this developer prompt [prompt] and identify missing inputs that would prevent a high-quality answer."
10. "Write a prompt that forces the model to compare implementation options before recommending one."

## Senior Developers

1. "Help me improve this architecture prompt [prompt] so it asks for options, trade-offs, risks, migration, and operational readiness."
2. "Rewrite this RFC prompt [prompt] to produce a structured decision document with context, goals, non-goals, alternatives, and rollout."
3. "Create a reusable prompt template for evaluating whether to adopt [technology] in an existing stack."
4. "Improve this incident-analysis prompt [prompt] so it separates facts, hypotheses, contributing factors, root causes, and action items."
5. "Turn this vague architecture request [request] into a precise prompt with system context, constraints, quality attributes, and decision criteria."
6. "Create a prompt that helps prioritize technical debt by customer impact, operational risk, developer productivity, and effort."
7. "Revise this migration planning prompt [prompt] so it includes phases, rollback, observability, data validation, and stakeholder communication."
8. "Create a prompt template for running an architecture review across scalability, security, reliability, cost, and maintainability."
9. "Evaluate this senior engineering prompt [prompt] and recommend stronger context, constraints, and success criteria."
10. "Write a prompt that asks the model to critique a technical strategy before proposing a roadmap."

## Product People

1. "Help me improve this product prompt [prompt] so it includes user, problem, evidence, goals, constraints, metrics, and decision needed."
2. "Rewrite this PRD prompt [prompt] to produce a clear document with problem, users, scope, non-goals, success metrics, risks, and open questions."
3. "Create a reusable prompt template for turning customer feedback into themes, insights, opportunities, and prioritized backlog items."
4. "Improve this roadmap prompt [prompt] so it connects bets to outcomes, confidence, dependencies, and trade-offs."
5. "Turn this vague product request [request] into a precise prompt with audience, context, evidence, constraints, and output format."
6. "Create a prompt that helps evaluate whether to build, delay, or reject [feature] using explicit decision criteria."
7. "Revise this experimentation prompt [prompt] so it includes hypothesis, target users, success metric, guardrails, and decision rule."
8. "Create a prompt template for writing launch plans that cover product, engineering, marketing, sales, support, legal, and analytics."
9. "Evaluate this product prompt [prompt] and identify missing research, assumptions, metrics, or constraints."
10. "Write a prompt that makes the model challenge a product strategy before generating recommendations."
